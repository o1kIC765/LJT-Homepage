This paper discovers a universal truthfulness hyperplane in large language models (LLMs) that can reliably distinguish truthful from false statements. The finding provides new insights into the internal mechanisms of LLMs and opens up possibilities for improved truthfulness control.

Co-authors: Shiqi Chen, Yu Cheng, Junxian He.

GitHub code repository available: Universal_Truthfulness_Hyperplane.
