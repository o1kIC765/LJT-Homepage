This work identifies and analyzes the perception bottleneck in vision-language models (VLMs) when applied to chart understanding tasks. We conduct extensive experiments to understand how VLMs process visual information from charts and propose mitigation strategies.

Co-authors: Weihao Zeng, Xiwen Zhang, Yijun Wang, Zifei Shan, Junxian He.

GitHub code repository available: Vision4Chart.
